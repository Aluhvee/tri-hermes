README
======
Site is setup using vuepress v2. https://v2.vuepress.vuejs.org/

1. initial setup - install nvm from https://github.com/coreybutler/nvm-windows (or use the og linux version)
```
nvm install 16
nvm use 16
npm install
```
2. running local server (logs should give you the url to the server)
```
npm run dev
```
3. testing out the github pages build (build files can be seen in `.vuepress/dist`)
```
npm run build
```
